### 注意事项

- 这个版本的修改bar的color稍微有点问题 但是不大。按照lore的模板将color修改即可，否则切换一个玩家的bar颜色会同时更改全局玩家bar的颜色
- 关于lore的修改请以res\assets\ui\textures\lang内的图片为准。需要注意的是图片高度必须为255，且图片左右两侧各有一个alpha很小的色块，目的是保持图片的宽度
